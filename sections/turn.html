Why I built it
Causal questions are easy to answer badly. Use the wrong method and you still get a clean coefficient, a p-value, a confidence interval. Nothing in the output says it’s wrong.

A trained researcher would draw a DAG, check the assumptions, and run a sensitivity analysis before trusting an estimate. I wanted to see how much of that care could be encoded in a system that doesn’t skip steps — so I built one, and had it check its own work.
