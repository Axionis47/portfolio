Causal Inference Agent

Thirteen specialist agents run the steps a careful analyst would — method selection, confounder identification, sensitivity analysis — coordinated by a ReAct orchestrator. An advocate, a skeptic, and a methodologist review each result against the actual data and can send it back to be redone. You give it a dataset; it returns a notebook you can run and check.
